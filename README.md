# Williams_Sarah_Capstone
The Cookie Girl Bakery Website:
This is a website developed for a web development class
The website was designed for an at-home bakery where people can order from common items or make custom orders and view recipies. The website has shopping cart abilities, form abilities, images, tables, and a slideshow. 

The website has the pages:
Home
About
Contact
Terms
Privacy
Recipies
Order
Cart

How to run project: 
Download the project files and open "index.html" 

Notes:
This is a student project and cannot be used to actually order any projects or process any payments.
